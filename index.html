Image Search Abstraction Layer

User stories:

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding the offset parameter to the URL
  3. I can get a list of the latest submitted search strings.

Example usage:

https://immense-dusk-62711.herokuapp.com/search/lolcats%20funny
